مميزات الخدمة

منيو أونلاين

  • عرض كل المنتجات بالصور والأسعار
  • تصنيف المنتجات بسهولة
  • تجربة مستخدم سلسة على الموبايل والكمبيوتر
  • تجربة مجانية لمدة أسبوع قبل الاشتراك

لوحة تحكم سهلة

  • تعديل الأسعار والصور بسهولة
  • إضافة/حذف منتجات
  • متابعة المنتجات الأكثر طلبًا

QR Code جاهز

  • يمكن طباعته للطاولات أو البوسترات
  • متوافق مع أي موبايل
  • تجربة سريعة للعملاء

الباقات والأسعار

Starter

800 – 1000 جنيه/سنة

  • منيو إلكتروني أساسي
  • QR Code جاهز للطباعة
  • لوحة تحكم محدودة (5 منتجات)

Standard

1200 – 1500 جنيه/سنة

  • منيو كامل وعدد غير محدود من المنتجات
  • QR Code لكل طاولة أو عام
  • لوحة تحكم سهلة + دعم فني أساسي

Premium

1800 – 2500 جنيه/سنة

  • كل مزايا Standard
  • تصميم منيو مخصص
  • تحديث غير محدود + دعم سريع
